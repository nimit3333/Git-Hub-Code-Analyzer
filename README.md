🚀 AI Code Analyzer – Understand GitHub Code Instantly

AI Code Analyzer is a Chrome Extension + Backend system that helps you quickly understand any code on GitHub using AI.

With a single click, it analyzes the code and provides:
• Simple explanation of what the code does  
• Step-by-step logic breakdown  
• Bugs and bad practices  
• Suggested improvements  
• Performance optimizations  

💡 Perfect for:
• Students learning new code  
• Developers exploring open-source projects  
• Anyone who wants faster code understanding  

⚙️ How it works:
1. Open any GitHub repository file  
2. Click "Analyze Code"  
3. Instantly get AI-powered insights  

🔒 Privacy:
Your code is securely sent to a backend for analysis and is not stored or shared.

🛠 Tech Stack:
• Java (Spring Boot backend)  
• Chrome Extension (JavaScript, CSS)  
• Gemini AI API  

🚀 Future Improvements:
• Support for LeetCode & other platforms  
• Faster responses with caching  
• Better UI and code highlighting
