AI Code Analyzer does not collect, store, or share any personal user data.

The extension only processes the code that the user explicitly chooses to analyze. This code is sent securely to a backend service for processing and is not stored permanently.

No personally identifiable information (such as name, email, or browsing history) is collected.

The extension does not track user activity outside of the functionality required to analyze code on GitHub.

All communication between the extension and backend services is done securely.

If you have any questions or concerns, please contact via the GitHub repository.
